# Cloud Fun Facts Generator ☁️

A simple serverless web application that displays fun facts about cloud computing.

## Features

- Fetches cloud facts from a serverless API
- Clean and responsive UI
- One-click fact generation

## Tech Stack

- AWS Lambda
- API Gateway (HTTP API)
- HTML, CSS, JavaScript
- GitHub
- AWS Amplify (for frontend hosting)

## API Endpoint

GET /funfact

## How It Works

1. User clicks "Generate Fun Fact"
2. Frontend calls API Gateway
3. API Gateway invokes Lambda
4. Lambda returns a cloud fact as JSON
5. Frontend displays the result
